# 本地以图搜图软件xiSearch

## 简介

项目地址：https://github.com/xjhqre/xiSearch/

## 使用说明

1、启动程序，进入设置页面，设置特征文件保存地址

![image-20240306212658444](README.assets/image-20240306212658444.png)

2、进入特征提取页面，输入要提取的图片库地址，点击特征提取按钮

![image-20240306212746955](README.assets/image-20240306212746955.png)

3、进入图片搜索页面搜索图片，点击图片即可复制文件路

![image-20240306212634741](README.assets/image-20240306212634741.png)
